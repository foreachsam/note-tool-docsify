
# 佈景主題

## 簡易設定

舉例，要設定「dark」這個佈景主題，只要將下面這一行加到「index.html」，就可以了。

``` html
<link rel="stylesheet" href="//unpkg.com/docsify/lib/themes/dark.css">
```

## 列表

預設的佈景主題，可以從下面兩個網址找到。

* https://unpkg.com/docsify/lib/themes/
* https://unpkg.com/docsify/themes/
